Who are they?

Send2Fax is a service that lets users send and receive faxes via email and the web.  They are now part of j2 Cloud Services.

What did I do, when, where?

In August of 2003, I created a program for them to initialize and test a Brooktrout TR114 fax board.  I also configured their Windows 2000 Server to run the program upon startup.  The work only took about ten hours, and was 100% remote.  I wrote it in C (yes, plain old C, not C++, C#, nor Objective-C), which was my primary language at that time.
